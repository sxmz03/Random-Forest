# Traffic Congestion Prediction

Random Forest classifier built on the Kaggle Traffic Prediction dataset. Takes vehicle counts (cars, bikes, buses, trucks) plus time and day-of-week features and classifies traffic as low, normal, high, or heavy.

Ordinal encoding for the categorical time/day features, 60/40 train-test split. Evaluation plots done with Seaborn and Matplotlib.

Dataset: [Kaggle — Traffic Prediction Dataset](https://www.kaggle.com/datasets/hasanramezani/traffic-prediction-dataset)
